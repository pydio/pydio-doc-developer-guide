## ./cells-enterprise admin user create

### Synopsis


DESCRIPTION

  Create a user

  Please, note that the login is case sensitive:

  You can also create a user in a given group by entering a full path
  (see examples below). 

EXAMPLES

  $ ./cells-enterprise user create -u '/group/user' -p 'a password'
