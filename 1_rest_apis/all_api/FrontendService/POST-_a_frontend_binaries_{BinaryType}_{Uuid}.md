






 
Upload frontend binaries (avatars / logos / bg images)  


### Body Parameters

Name | Description | Type | Required
---|---|---|---
**BinaryType** | Currently supported values are USER and GLOBAL | _string_ |   
**Uuid** | Id of the binary | _string_ |   


### Body Example
```
{
  "BinaryType": "string",
  "Uuid": "string"
}
```
