






 
Add a new value to Tags for a given namespace  


### Body Parameters

Name | Description | Type | Required
---|---|---|---
**Namespace** | Add a tag value for this namespace | _string_ |   
**Tag** | New tag value | _string_ |   


### Body Example
```
{
  "Namespace": "string",
  "Tag": "string"
}
```

{

}
